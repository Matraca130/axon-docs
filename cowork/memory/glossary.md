# Glossary — Axon Project

## Términos técnicos
| Term | Meaning |
|------|---------|
| FSRS v4 | Free Spaced Repetition Scheduler v4 — algoritmo de repetición espaciada |
| BKT v4 | Bayesian Knowledge Tracing v4 — modelado del conocimiento del estudiante |
| RAG | Retrieval-Augmented Generation — IA con contexto de docs del curso |
| pgvector | Extensión PostgreSQL para embeddings vectoriales (1536d) |
| CCA | Estructura de landing: Hero→Trust→Problema→Features→SocialProof→Pricing→CTA |
| spritesheet WebP | Técnica para scroll-linked video: frames como imagen WebP animada |
| Bento Grid | Layout de cards tipo grid para la landing |
| Versión B / C | Iteraciones de diseño de la landing page |
| Mux | Plataforma de video (upload + signed playback + tracking) |
