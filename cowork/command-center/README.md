# Command Center — PROJETO DESAROLLO RESUMEN

Version anterior del command center del proyecto Axon (pre-Cowork). Contiene la historia completa de planificacion, arquitectura, y decisiones del desarrollo del sistema de resumenes IA.

## Contenido principal

- **ARCHITECTURE.md** — Arquitectura del pipeline de resumenes
- **STRATEGIC_PLAN.md** — Plan estrategico del proyecto
- **DOCUMENTATION_MANIFEST.md** — Indice de toda la documentacion
- **Arquitectura_Pipeline_v2.jsx** — Prototipo visual del pipeline

### Subcarpetas (contenido aplanado aqui)
- agents/ — Registry de agentes original
- changelog/ — Changelog del proyecto
- checkpoints/ — Sesiones y checkpoints de trabajo
- docs/ — API reference, conventions, project overview, tech stack
- ideas/ — Ideas e implementaciones pendientes
- memory/ — Contexto, decisiones, learnings
- roadmap/ — Roadmap y milestones
- supabase/ — Schema SQL y sync guide

## Por que existe esto
Este fue el primer intento de organizar el proyecto antes de migrar a Cowork + .auto-memory. Parte de esta informacion ya esta consolidada en los docs actuales, pero se preserva como registro historico.
